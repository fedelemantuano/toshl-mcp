# Toshl MCP

**Author:** Fedele Mantuano <mantuano.fedele@gmail.com>

## Requirements

- Python 3.13 or 3.14
- [uv](https://docs.astral.sh/uv/)

## Installation

```bash
make install
```

## Development

```bash
make lint        # check code
make lint-fix    # check and auto-fix
make format      # format code
make test        # run tests
make pre-commit  # run all pre-commit hooks
```
